# ATTClassificationTool
An automatic tickets classification tool uses LLM by Langchain.
## Cofiguration

Setting up variables (.env)
```
MODEL_CHATGPT="all-MiniLM-L6-v2"
TRAINED_MODEL="modelsvm.pk1"
Pinecone_API_Key=" "
PINECONE_ENVIRONMENT=" "
PINECONE_INDEX=" "
OPENAI_API_KEY=" "
```
